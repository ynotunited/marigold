# 16. Design System

## Overview

The Marigold Commerce Platform should present itself as a premium digital experience that inspires confidence from the very first interaction.

The design language should be:

- Minimal
- Elegant
- Modern
- Confident
- Product-first
- Motion-enhanced
- Accessible

The interface should never feel cluttered or overly decorative.

Every element should have purpose.

---

# Design Principles

## 1. Premium Before Colour

Premium does not come from using gold.

Premium comes from:

- spacing
- typography
- photography
- hierarchy
- consistency

Gold is only an accent.

---

## 2. Products Are The Hero

The website exists to sell products.

The interface should support the products instead of competing with them.

Large product photography.

Clean backgrounds.

Very little unnecessary decoration.

---

## 3. Motion Should Guide

Animations should direct attention.

Never distract.

Every animation should answer one question:

"What is the user supposed to notice?"

---

## 4. Consistency

Buttons behave the same.

Cards behave the same.

Forms behave the same.

Spacing follows one system.

Animations follow one system.
