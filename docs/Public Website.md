# Homepage

## Objective

The homepage should:

- Introduce the Marigold brand.
- Build trust within the first 10 seconds.
- Showcase products without overwhelming visitors.
- Encourage purchases.
- Encourage quotation requests.
- Inspire confidence in corporate buyers.

The homepage should tell a story.

It should never feel like an online marketplace.
